Self-Introduce

1년 6개월 동안 몸 담고 있었던 곳들에서 수행한 업무에 대한 대략적인 진술입니다.

주요 업무

주 업무는 HTML과 CSS를 사용한 웹사이트 제작 및 javascript 와 jQuery 를 통한 동적인 작업을 처리했습니다.

사용 툴

1년 까지 nineonelabs 에 있을 때는 EDITPLUS 를 활용했었다가 메타올에 있을 때는 VISUAL STUDIO CODE를 사용하여 작업을 했습니다.

반응형

일을 시작하고 2달 가량은 적응형으로 작업을 처리했었는데, 개발 코드를 입히거나 수정 작업을 할 때 2번 적용해야하는 번거로움 때문에 2달 정도가 지나서 반응형으로 작업 방식을 변경했습니다. 반응형 작업에서 max-width 값은 기기의 해상도에 맞췄으며, 간혹 디자인이 어긋나는 경우에는 값을 세부적으로 주어서 세세하게 수정했습니다. 그리고 추후에는 768px 아래로는 css viewport 를 활용해서 일관된 레이아웃이 구성되도록 작업을 진행했습니다.

크로스 브라우징

나인원랩스에서는 9개월 가량은 레이아웃을 정렬할 때 float 나 inline-block를 사용했었습니다. css flex 나 grid를 사용했던 때는 올 해 2월, 그러니까 3개월 전 부터입니다. 원래 요청은 인터넷 익스플로러 사용 유저를 고려해서 항상 float 나 grid를 사용하지 않았었습니다. 그런데 아시다시피 flex 같은 코드가 내용이 유동적으로 변해야하는 경우 height 값을 따로 주지 않아도 된다는 편의성 뿐만아니라 시각적으로도 깔끔한 디자인을 고수할 수 있어서, 당시에 업체의 특성 및 사용 유저의 특성에 따라 정렬 방식을 다르게 하는 것이 좋을 것 같다고 건의를 드렸었습니다. 그래서 대체로 나이가 드신 분들이 주 고객층인 업체의 경우 float 나 inline-block를 사용했으며, 아닌 경우에는 flex 코드를 사용하여 작업을 진행했습니다.

메타올에서는 업체 특성 상 인터넷 익스플로러에 신경을 쓰지 않아 크로스 브라우징 작업이 별로 없었습니다. 그래서 주로 FLEX 를 활용해서 작업을 진행했으며 간혹 IOS 기기 때문에 크로스브라우징 작업을 진행했습니다.
